# Cabrillo Cafe System

A web-based cafe management system.

The project allows customers to view menu items and place orders while the cafe can manage items and orders.

## Tech Stack

Frontend
- React
- TypeScript
- CSS

Backend
- Node.js
- Express

Database
- (PostgreSQL / MongoDB / Firebase - choose later)

## Features (Planned)

- View cafe menu
- Place orders
- Admin panel for managing menu items
- Order tracking

## Installation

Clone the repository

git clone https://github.com/Emir7530/cabrillo-cafe

Go to backend folder

cd backend

Install dependencies

npm install

Start the server

npm run dev
